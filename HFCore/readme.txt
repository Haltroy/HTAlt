Thanks for using the HaltroyFramework.
Note that this project is still in development
and bugs can be visible.If you spot an one
then please create an issue on GitHub.

To use HaltroyFramework, plase make sure you
have at least .Net Framework 2.0 installed
on your machine.Newer versions are supported.

Users of your projects are also need to have
at least .Net Framework 2.0 or newer.

If you are using this then please take screenshot
and send this to Haltroy if you want to get a free
shoutout.

The social links are located bottom of the website.
http://haltroy.com

Current Features:
-Tab Control
-Slider Control
-ListView Control
-MessageBox
-InputBox
-Switch Control

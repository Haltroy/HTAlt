Thanks for using HT-Alt.
Note that this project is still in development. If you have problems then please create an 
issue on GitHub so I can fix it.

If you are getting write errors while building after updating, please close Visual Studio and open it
again. If this didn't worked, feel free to open an issue on GitHub.
https://github.com/Haltroy/HTAlt/issues/

To use HT-Alt, please make sure you have at least .Net Framework 4.5 installed
on your machine. Newer versions are supported.

Users of your projects are also need to have at least .Net Framework 4.5 or newer installed
on their machine too.

If you need help for something, you can create issues on GitHub page or you can check out the Wiki pages
on Github page. Examples for the features also available in "HTAlt.Test" project in solution file on GitHub page.
https://github.com/Haltroy/HTAlt

Current Features:
-Tab Control
-Slider Control
-Button Control
-ListView Control
-Switch Control
-HaltroyForms Form Type
-MessageBox Form
-InputBox Form
